## 유용한 VSCODE 단축키 모음

**마크다운 미리보기** : `ctrl` + `shift` + `V`  
**다중 라인 편집** : `ctrl` + `alt` + `아래 화살표`  
**동일 변수명 모두 편집** : 변수 클릭 후 `F2`  
**이모지 추가하기** : 윈도우 키 + `:`
