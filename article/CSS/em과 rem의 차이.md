EM과 REM의 차이
===

![em](https://user-images.githubusercontent.com/43127789/100442963-37b5ab80-30ec-11eb-8afb-b8ce273041d1.PNG)
```html
<div class="level1">
  <h1>level 1</h1>
  <div class="level2">
    <h1>level 2</h1>
    <div class="level3">
      <h1>level 3</h1>
      <div class="level4">
        <h1>level 4</h1>
      </div>
    </div>
  </div>
</div>
```
```css
.level1,
.level2,
.level3,
.level4 {
  font-size: 2rem;
}
* {
 margin: 0;
 padding: 0;
}
```
